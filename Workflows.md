# Workflows

## Claude Code + GitHub

**Steps**

1. Create a GitHub account for yourself.
2. Create a GitHub account for Claude Code.
3. Create a public repository with your GitHub account.
4. Fork the repository with Claude Code's GitHub account.
5. Go to https://claude.ai/code/ and every time you want to give Claude Code a
task, create a new session in its forked repository. Claude Code will naturally
create git branches for every task.
6. Whenever you are satisfied with Claude Code's result in one of its branches:
    - Create a pull request in your personal repository for that branch
    - Accept and merge the pull request
    - Synchronize Claude Code's repository to your updated personal repository

**Advantages**
- Only requires a web browser
- Very secure as long as you don't give any secret to Claude Code
